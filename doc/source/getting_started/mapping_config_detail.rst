映射配置
=========================

为了进一步探索不同核心的任务映射与排布，对于模型部署的影响， **NPU-SIM** 支持在 :doc:`workload_config_detail` 的基础上进一步使用映射配置定义核心的重映射。

我们提供了示例配置文件，位于以下目录中：

.. code-block:: console

    ${NPU_SIM_ROOT}/llm/test/mapping_config

书写方法
------------

每一行代表一个重映射规则，使用半角冒号分隔前后两个数字。`A : B`即代表将核心A映射至核心B。

.. note:: 

    需保证若任意一个核心编号出现在配置中，那么该编号需在所有行的冒号左侧和右侧各出现、且仅出现一次。
